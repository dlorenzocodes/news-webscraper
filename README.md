# Project Title

Webscrapper - News

# Description

This program scrapes the USAToday website to get latest news. This is entirely for personal use. 

After it gathers the news from the home page, it sends an email to the specified email account. 

It uses OAuth2 to use gmail account. For this, OAuth consent screen and credentials will need to be generated and stored in an .env file. 

More information about OAuth is on nodemailer documantation. 

![alt tag](assets/email.png "Email Template")

# Installation 

1. Use ```git clone``` to copy repository
2. Create a ```.env``` file to store credentials
3. Use ```npm install``` to install all dependencies 
4. Start server by running ```npm start```
